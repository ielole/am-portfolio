About Me.
 

My enthusiasm for coding began with a book. As an avid reader, this wasn't a big surprise, but my immediate desire to write lines of code and the anticipation of viewing that code as it was interpreted by a browser was shocking. While attempting to write a Rails app to handle my cherished book list, I got a brief glimpse into the vast knowledge base that is web development and had two realization. First, I realized that I had found a career path that I could see myself pursing for the next 30 years. The second realization was that I would need professional instruction to accomplish my goal of becoming the kind of web developer that I was envisioning.
